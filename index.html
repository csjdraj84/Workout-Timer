How the Workout Timer works

  • 1. Configure name, number of workout types, repetitions per type, work time, and rest time.
  • 2. Each workout type runs its repetitions automatically: WORK → REST → WORK → REST…
  • 3. Last workout has no trailing rest. Last 5 seconds of every WORK and REST interval triggers audio beep + spoken countdown.
  • 4. Fully mobile responsive. Save / download / load configs as JSON. All data stored in browser.
